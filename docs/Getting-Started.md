# Getting Started

## Install

- **CurseForge client** — search for "Wick's Quest Key".
- **Manual** — download the latest ZIP from [CurseForge](https://www.curseforge.com/wow/addons/wicks-quest-key) or [GitHub Releases](https://github.com/Wicksmods/WicksQuestKey/releases), extract the `WicksQuestKey` folder into `World of Warcraft\_classic_\Interface\AddOns\`.

## First run

1. Pick up a quest that gives you a usable item (most "use this item on X" quests do).
2. The button appears on screen showing the icon of the quest item.
3. `/wqk` to confirm what is loaded.

## Bind a key

1. Open *Esc → Key Bindings*.
2. Scroll to the **Wick's Quest Key** section near the bottom.
3. Click the slot next to *Use current quest item* and press the key you want.
4. The bind label appears on the button immediately.

That's it. From now on, that key fires whichever quest item is armed; press again to advance to the next.

## Moving the button

- `/wqk unlock` — unlocks dragging
- left-click drag the button anywhere on screen
- `/wqk lock` — re-locks it

Position is saved per character. `/wqk reset` returns it to the default location.

## Multiple quest items

If you have several active quest items at once, the button cycles through them on each press. Right-click to skip without firing — useful if the wrong item is armed and you don't want to waste a charge to advance.
