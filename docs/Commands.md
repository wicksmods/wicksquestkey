# Slash Commands

| Command | Effect |
|---|---|
| `/wqk` | List currently loaded quest items |
| `/wqk unlock` | Unlock the button so it can be dragged |
| `/wqk lock` | Lock the button back into place |
| `/wqk reset` | Reset position to default (center, lower-middle) |
| `/questkey` | Same as `/wqk` (long-form alias) |

## In-game actions

- **Left-click** the button — fires the armed quest item, then cycles to the next.
- **Right-click** the button — cycles to the next item without firing.
- **Hover** — shows the standard item tooltip.

## Key binding

Bind a key under *Esc → Key Bindings → Wick's Quest Key → Use current quest item*. The chosen key shows up in the top-right corner of the button.
